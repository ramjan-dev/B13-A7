Project Name

KeenKeeper — Keep Your Friendships Alive

Short Description

KeenKeeper is a modern communication and friendship management web app built with Next.js. It helps users stay connected with their friends by tracking interactions like calls, texts, and video chats. The app is fast, responsive, and designed with a clean user experience, making it easy to maintain strong relationships and never lose touch.

Technologies Used
Next.js
React.js
React Router DOM
Tailwind CSS
daisyUI
Recharts
React Hot Toast
Key Features

🔹 Friend Dashboard
All friends are displayed in a card-based layout with their status, tags, and the number of days since last interaction.

🔹 Interaction Timeline
Automatically logs every Call, Text, and Video activity with date and type, creating a realistic and dynamic timeline.

🔹 Real-Time Interaction Simulation
Generates real-time data when users click on call, text, or video, using React Context API for smooth state management.

🔹 Category-Based Filtering
Easily filter interactions by calls, texts, or videos to view specific activity types.

🔹 Friendship Analytics
Visual pie charts powered by Recharts to analyze interaction patterns and communication habits.

🔹 Smart Reminder Concept
Helps users remember to reach out to friends before relationships become inactive.
